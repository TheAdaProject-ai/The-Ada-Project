# Ada.Chat - Chat-Focused AI Assistant

Ada.Chat is a streamlined version of Ada focused on conversational AI interactions. She provides intelligent responses using multiple free APIs in a clean, distraction-free interface.

## Features

- **💬 Chat Interface**: Clean, modern chat interface with intelligent responses
- **🔍 Multiple Search APIs**: Wikipedia, DuckDuckGo, Dictionary, and more
- **🌤️ Weather Modal**: Current weather information via Open-Meteo API
- **🧮 Calculator Modal**: Mathematical calculations with support for percentages, exponents, and trigonometry
- **🌐 Translator Modal**: Translate text to 14 different languages
- **🌙 Dark Mode**: Toggle between light and dark themes
- **📝 Conversation Management**: Save, delete, and download conversations
- **😀 Reactions**: React to messages with emojis

## API Integrations

- **Dictionary API**: Word definitions and synonyms
- **Wikipedia API**: Article summaries with fuzzy matching
- **DuckDuckGo API**: Web search capabilities
- **Open-Meteo API**: Weather information
- **MyMemory Translation API**: Translation services

## How It Works

Ada.Chat uses a multi-layered approach to answer questions:

1. **Greeting Detection**: Recognizes and responds to common greetings
2. **Weather Lookup**: Checks Open-Meteo API for weather queries
3. **Calculator**: Evaluates mathematical expressions
4. **Translation**: Uses MyMemory API for translation requests
5. **Dictionary Lookup**: Checks for word definitions
6. **Wikipedia Search**: Uses MediaWiki REST API with fuzzy matching
7. **DuckDuckGo Fallback**: Searches the web if Wikipedia fails

## Setup

1. Clone or download this repository
2. Open `Ada_Chat.html` in a modern web browser (Chrome, Edge, Firefox, Safari)
3. Ada.Chat is ready to use immediately

## Usage

### Chat
- "Hi", "Hello", "How are you?"
- "Define serendipity"
- "Who is Barack Obama?"
- "What is the weather in London?"
- "Calculate 2^10"
- "Translate hello to French"

### Modals
- **Weather**: Click 🌤️ button, enter city name
- **Calculator**: Click 🧮 button, enter expression
- **Translator**: Click 🌐 button, enter text and select language

### Dark Mode
- Click 🌙/☀️ button to toggle

## Browser Support

- **Chrome/Edge**: Full support
- **Firefox/Safari**: Full support
- **Modern browsers**: All features supported

Modify the JavaScript in `Ada_Chat.html` to customize:
