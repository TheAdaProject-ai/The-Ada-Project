# Ada.Mini - Compact AI Assistant

Ada.Mini is a lightweight, compact version of Ada designed for quick interactions and minimal resource usage. She provides intelligent responses using multiple free APIs in a streamlined interface.

## Features

- **💬 Chat Interface**: Compact, efficient chat interface
- **🔍 Multiple Search APIs**: Wikipedia, DuckDuckGo, Dictionary
- **🌤️ Weather Modal**: Current weather information via Open-Meteo API
- **🧮 Calculator Modal**: Mathematical calculations
- **🌐 Translator Modal**: Translate text to 14 different languages
- **🌙 Dark Mode**: Toggle between light and dark themes
- **📝 Conversation Management**: Save, delete, and download conversations

## Design Philosophy

Ada.Mini is designed for:
- **Speed**: Fast loading and response times
- **Simplicity**: Clean, minimal interface
- **Efficiency**: Optimized for quick interactions
- **Portability**: Works well on all devices

## API Integrations

- **Dictionary API**: Word definitions and synonyms
- **Wikipedia API**: Article summaries with fuzzy matching
- **DuckDuckGo API**: Web search capabilities
- **Open-Meteo API**: Weather information
- **MyMemory Translation API**: Translation services

## How It Works

Ada.Mini uses a streamlined approach to answer questions:

1. **Greeting Detection**: Recognizes and responds to common greetings
2. **Weather Lookup**: Checks Open-Meteo API for weather queries
3. **Calculator**: Evaluates mathematical expressions
4. **Translation**: Uses MyMemory API for translation requests
5. **Dictionary Lookup**: Checks for word definitions
6. **Wikipedia Search**: Uses MediaWiki REST API with fuzzy matching
7. **DuckDuckGo Fallback**: Searches the web if Wikipedia fails

## Setup

1. Clone or download this repository
2. Open `Ada_Mini.html` in a modern web browser (Chrome, Edge, Firefox, Safari)
3. Ada.Mini is ready to use immediately
