# Ada.Coffee - Coffee-Themed AI Assistant

Ada.Coffee is a themed version of Ada with a coffee-inspired design and personality. She combines multiple free APIs to provide intelligent responses in a cozy, cafe-style interface.

## Features

- **💬 Chat Interface**: Coffee-themed chat interface with intelligent responses
- **☕ Coffee Personality**: Specialized responses for coffee-related queries
- **🔍 Multiple Search APIs**: Wikipedia, DuckDuckGo, Dictionary, and more
- **🌤️ Weather Modal**: Current weather information via Open-Meteo API
- **🧮 Calculator Modal**: Mathematical calculations
- **🌐 Translator Modal**: Translate text to 14 different languages
- **🌙 Dark Mode**: Toggle between light and dark themes
- **📝 Conversation Management**: Save, delete, and download conversations

## Specialized Features

### Coffee Knowledge
- Coffee bean varieties and origins
- Brewing methods and techniques
- Coffee culture around the world
- Cafe recommendations and tips

### Coffee Personality
- Friendly, cafe-style conversational tone
- Coffee-themed greetings and responses
- Cozy, warm interface design

## API Integrations

- **Dictionary API**: Word definitions
- **Wikipedia API**: General knowledge lookup
- **DuckDuckGo API**: Web search capabilities
- **Open-Meteo API**: Weather information
- **MyMemory Translation API**: Translation services

## How It Works

Ada.Coffee uses a multi-layered approach with a coffee-themed personality:

1. **Greeting Detection**: Coffee-themed greetings
2. **Coffee Queries**: Specialized responses for coffee-related topics
3. **General Knowledge**: Uses Wikipedia and DuckDuckGo for other queries
4. **Calculator**: Evaluates mathematical expressions
5. **Translation**: Uses MyMemory API for translation requests

## Setup

1. Clone or download this repository
2. Open `Ada_Coffee.html` in a modern web browser (Chrome, Edge, Firefox, Safari)
3. Ada.Coffee is ready to use immediately

## Usage
