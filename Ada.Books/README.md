# Ada.Books - Book-Focused AI Assistant

Ada.Books is a specialized version of Ada optimized for book-related queries and literary discussions. She combines multiple free APIs to provide comprehensive information about books, authors, literature, and related topics.

## Features

- **💬 Chat Interface**: Clean, modern chat interface with intelligent responses
- **📚 Book Search**: Search for books from Project Gutenberg via Gutendex API
- **🔍 Wikipedia Integration**: Look up authors, literary movements, and literary terms
- **🌤️ Weather Modal**: Current weather information via Open-Meteo API
- **🧮 Calculator Modal**: Mathematical calculations with support for percentages, exponents, and trigonometry
- **🌐 Translator Modal**: Translate text to 14 different languages
- **🌙 Dark Mode**: Toggle between light and dark themes
- **📝 Conversation Management**: Save, delete, and download conversations
- **🎓 Literary Knowledge**: Specialized responses for literature-related queries

## Specialized Features

### Book Search
- Search Project Gutenberg by title, author, or subject
- Get author information and book metadata
- Download counts and popularity data
- Subject category filtering

### Literary Queries
- Author biographies and literary analysis
- Book summaries and plot information
- Literary movement and genre information
- Historical context for literary works

## API Integrations

- **Gutendex API**: Project Gutenberg book search
- **Wikipedia API**: Author and literary information
- **Dictionary API**: Word definitions and literary terms
- **Open-Meteo API**: Weather information
- **MyMemory Translation API**: Translation services
- **DuckDuckGo API**: Web search fallback

## How It Works

Ada.Books uses a specialized approach for book-related queries:

1. **Book Detection**: Recognizes book-related queries (titles, authors, genres)
2. **Gutendex Search**: Searches Project Gutenberg for matching books
3. **Author Lookup**: Uses Wikipedia for author biographies
4. **Literary Context**: Provides literary analysis and historical context
5. **Fallback APIs**: Uses Wikipedia and DuckDuckGo for general queries

## Setup

1. Clone or download this repository
2. Open `Ada_Books.html` in a modern web browser (Chrome, Edge, Firefox, Safari)
3. Ada.Books is ready to use immediately
