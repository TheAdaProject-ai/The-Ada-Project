# Ada.Students - Student-Focused AI Assistant

Ada.Students is a specialized version of Ada optimized for educational queries and student needs. She combines multiple free APIs to provide comprehensive information for learning, homework help, and academic research.

## Features

- **💬 Chat Interface**: Clean, modern chat interface with intelligent responses
- **🎓 Educational Support**: Specialized responses for student queries
- **🔍 Multiple Search APIs**: Wikipedia, DuckDuckGo, Dictionary, and more
- **🌤️ Weather Modal**: Current weather information via Open-Meteo API
- **🧮 Calculator Modal**: Mathematical calculations with support for percentages, exponents, and trigonometry
- **🌐 Translator Modal**: Translate text to 14 different languages
- **🌙 Dark Mode**: Toggle between light and dark themes
- **📝 Conversation Management**: Save, delete, and download conversations

## Specialized Features

### Educational Queries
- Homework help and explanations
- Concept definitions and examples
- Study tips and learning strategies
- Academic subject knowledge

### Subject Support
- Mathematics (algebra, calculus, geometry)
- Science (physics, chemistry, biology)
- History and social studies
- Literature and language arts
- Computer science and programming

## API Integrations

- **Dictionary API**: Word definitions and synonyms
- **Wikipedia API**: Educational content and explanations
- **DuckDuckGo API**: Web search for research
- **Open-Meteo API**: Weather information
- **MyMemory Translation API**: Translation services
- **Gutendex API**: Book search for literature

## How It Works

Ada.Students uses a specialized approach for educational queries:

1. **Subject Detection**: Recognizes educational topics and subjects
2. **Concept Explanation**: Provides clear, educational explanations
3. **Homework Help**: Offers guidance on problem-solving
4. **Research Support**: Uses Wikipedia and DuckDuckGo for academic research
5. **Calculator**: Evaluates mathematical expressions
6. **Translation**: Uses MyMemory API for translation requests

## Setup

1. Clone or download this repository
2. Open `Ada_Students.html` in a modern web browser (Chrome, Edge, Firefox, Safari)
3. Ada.Students is ready to use immediately

## Usage

### Educational Queries
- "Explain photosynthesis"
- "What is the Pythagorean theorem?"
- "Help me with calculus"
- "Define literary devices"
- "Who was Abraham Lincoln?"

### Subject-Specific
- **Math**: "Calculate 2^10", "What is the derivative of x^2?"
- **Science**: "What is gravity?", "Explain the water cycle"
- **History**: "Causes of World War II", "Who was Napoleon?"
- **Literature**: "Themes of Romeo and Juliet", "Define metaphor"

### Modals
- **Weather**: Click weather button, enter city name
- **Calculator**: Click calculator button, enter expression
- **Translator**: Click translator button, enter text and select language
